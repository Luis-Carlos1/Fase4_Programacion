# Fase4_Programacion
Sistema de Gestión de Reservas - Software FJ Este proyecto es una aplicación de escritorio desarrollada en Python. El sistema nos permite gestionar clientes y servicios (Salas, Equipos y Asesorías) aplicando los pilares de la Programación Orientada a Objetos (POO).
# Estructura
- Registrar clientes.
- Gestionar diferentes tipos de servicios.
- Crear, confirmar y cancelar reservas.
- Validar datos ingresados.
- Manejar errores mediante excepciones personalizadas.
- Registrar eventos y errores en archivos de logs.
# Características
El sistema permite registrar clientes y guardar su información de manera organizada.

Se pueden crear diferentes tipos de servicios como reservas de salas, alquiler de equipos y asesorías.

El programa maneja errores utilizando excepciones para evitar que el sistema se cierre inesperadamente.

Todos los eventos importantes y errores quedan registrados en un archivo de logs para tener un mejor control del sistema.
# Requisitos
Python y libreria tinker
# Instrucciones
Ejecutar el archivo principal llamado main.py.

Utilizar el menú del sistema para registrar clientes, crear servicios y realizar reservas.

Revisar el archivo de logs para verificar eventos o errores registrados durante la ejecución del programa.
